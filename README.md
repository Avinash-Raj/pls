pls
==============

Python script to list files or folders according to the input regex.

```sh
$ python pls.py -h
usage: pls.py [-h] [-r] [-d] regex

Utility for listing files.

positional arguments:
  regex       Regex

optional arguments:
  -h, --help  show this help message and exit
  -r          Recursive search.
  -d          List Only directories.
```
